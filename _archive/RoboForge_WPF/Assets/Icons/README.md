Place custom vector graphics and SVG icons here.
Currently the UI uses highly scalable Unicode/FontAwesome-style text icons embedded directly into XAML strings. This guarantees crisp, artifact-free scaling at any DPI and minimizes memory usage, aligning with the glossy, modern Dark Industrial aesthetic requested.

If custom image assets (PNGs/SVGs) must be used in the future, drop them here and reference them as Image/Path resources in Styles.xaml.
